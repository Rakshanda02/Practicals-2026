<h2>Aim of the Practical</h2>
<p>
To develop a webpage to display tabular data such as a timetable or student
record using HTML advanced table tags and attributes.
</p>

<h2>Table Elements Used</h2>
<ul>
<li>&lt;table&gt; : Container for all table data.</li>
<li>&lt;tr&gt; : Defines a table row.</li>
<li>&lt;th&gt; : Defines a table header cell.</li>
<li>&lt;td&gt; : Defines table data.</li>
<li>&lt;caption&gt; : Title of the table.</li>
<li>&lt;thead&gt; : Groups header rows.</li>
<li>&lt;tbody&gt; : Groups body rows.</li>
<li>&lt;tfoot&gt; : Groups footer rows.</li>
<li>rowspan : Merges cells vertically.</li>
<li>colspan : Merges cells horizontally.</li>
</ul>
